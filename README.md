# Samsung AC Пульт (для Xiaomi/Redmi з ІЧ-передавачем)

ІЧ-пульт для кондиціонера **Samsung MH19ZC2-12** (оригінальний пульт ARH-428 / DB93-00251L).
Інтерфейс відтворює основні кнопки оригіналу, працює через апаратний ІЧ-передавач телефона.

Протокол передачі — **Samsung A/C**, портований із перевіреної бібліотеки
[IRremoteESP8266](https://github.com/crankyoldgit/IRremoteESP8266) (точні тайминги,
кодування стану та контрольні суми збережені без змін).

## Як отримати готовий APK без Android Studio (через GitHub Actions)

1. Створи безкоштовний акаунт на https://github.com і новий **порожній репозиторій**
   (наприклад `samsung-remote`), бажано **Private**.
2. Завантаж туди **вміст папки `SamsungRemote`** (усі файли й підпапки, включно з
   прихованою `.github`). Це можна зробити:
   - кнопкою **Add file → Upload files** на сайті GitHub (перетягни файли), або
   - через `git`:
     ```
     cd SamsungRemote
     git init
     git add .
     git commit -m "Samsung AC remote"
     git branch -M main
     git remote add origin https://github.com/USERNAME/samsung-remote.git
     git push -u origin main
     ```
3. Відкрий вкладку **Actions** у репозиторії. Збірка «Build APK» запуститься
   автоматично (або натисни **Run workflow**). Зачекай ~3–5 хв.
4. Коли з'явиться зелена галочка — відкрий запуск, унизу в розділі **Artifacts**
   завантаж архів **SamsungRemote-debug-apk**. Усередині — `app-debug.apk`.
5. Перекинь `app-debug.apk` на телефон і встанови (увімкни «Встановлення з
   невідомих джерел», якщо попросить).

> Альтернатива: відкрити папку `SamsungRemote` в **Android Studio** → `Build →
> Build APK(s)`. Android Studio сам догенерує Gradle Wrapper.

## Кнопки

| Кнопка | Дія |
|---|---|
| ЖИВЛЕННЯ | Увімк./вимк. |
| Темп − / + | Температура 16–30 °C |
| Режим | Охолодж. → Осушення → Вент. → Обігрів → Авто |
| Вентилятор | Авто → Низька → Середня → Висока → Турбо |
| Обдув | Хитання жалюзі (вкл/викл) |
| Турбо | Потужний режим |
| Тихо | Тихий режим |
| Підсвітка | Дисплей кондиціонера |
| Таймер ON / OFF | 0 → 1 → 2 → 3 → 6 → 9 → 12 год |
| Сон | Нічний режим (8 год) |
| Скинути | Початковий стан |

## Якщо кондиціонер не реагує

Модель MH19ZC2-12 — старіша, і її пульт ARH-428 теоретично може використовувати
дещо інший варіант ІЧ-кодування, ніж сучасний Samsung A/C, реалізований тут.
Якщо частина або всі кнопки не діють:

1. **Перевір сам ІЧ-передавач** стандартним застосунком (Mi Remote): якщо ним
   вдається керувати будь-якою технікою — залізо працює.
2. Якщо залізо працює, але цей кондиціонер не реагує — найімовірніше потрібні
   «рідні» коди саме цього пульта. Їх можна зчитати з оригінального пульта
   приймачем (наприклад, Arduino/ESP8266 з ІЧ-приймачем або Flipper Zero) і
   надіслати мені — я підставлю точні тайминги/байти, і пульт працюватиме 1-в-1.

Технічні деталі протоколу — у файлі
`app/src/main/java/com/redmi/samsungremote/SamsungAcProtocol.kt`.
